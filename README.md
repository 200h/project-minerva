# Project Minerva

Project Minerva is a Unity 5.6 project targeting Windows and iOS.

## Opening the Project

1. Install Unity 5.6.
2. In the Unity project selector, choose **Open**.
3. Select the repository root (the directory containing `Assets/`).

Project code uses `Minerva` as its root namespace.

See the [Project Constitution](Docs/Project/README.md) for the project’s governing documentation.
